## 密码隐藏显示动画

import PasswordDisplayAnimate from '@examples/snippit/password-display-animate.astro'

<PasswordDisplayAnimate></PasswordDisplayAnimate>

<details>
  <summary>查看说明</summary>

- CSS 样式：
  - 使用 CSS 变量和 calc() 函数来动态计算数字的放大和模糊程度。
  - 使用 :hover 和 :focus-visible 伪类来定义数字在悬停或聚焦时的样式变化。
  - 使用 filter 和 scale 属性来实现模糊和放大效果。
- CSS 选择器：使用 :is() 和 :has() 选择器来实现相邻数字的联动效果，允许根据当前数字的状态来影响后续数字的样式。
- CSS 动画：使用 transition 属性来平滑地过渡数字的放大和模糊效果，使得用户体验更加流畅。

</details>
